# Zaichyk Name Archive

This repository contains the static site files for [zaichyk.org](https://zaichyk.org), a source-based historical archive for the surname **Зайчик / Zaichyk / Zaichik**.

The archive documents the surname through primary and archival records, including Orthodox and Catholic church records, regional record clusters, early attestations, civic references, and historical context from Ukraine, Belarus, Volhynia, Kherson, Crimea, and related regions.

## Site

- Main site: https://zaichyk.org
- Surname page: https://zaichyk.org/surname/
- Earliest attestations: https://zaichyk.org/attestations/
- Parish records: https://zaichyk.org/attest/parish-records/
- Regional records: https://zaichyk.org/regions/
- Sources and method: https://zaichyk.org/sources/

## Purpose

The project treats surname history as a documentary record problem rather than a single origin claim. It focuses on where the surname appears, how it is written, what kind of institution recorded it, and what historical setting gives the record meaning.

## Notes

This repository contains the deployed static site output. Source materials, archival references, and page content are curated for presentation on the public site.
